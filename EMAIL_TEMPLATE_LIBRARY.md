- Single column layout
- Minimum 14px font size
- Tap-friendly CTA buttons (44px min height)
- Images load fast or degrade gracefully

## âœï¸ Copy Best Practices

### Subject Lines
- **Length:** 40-50 characters (mobile preview)
- **Avoid:** ALL CAPS, excessive punctuation (!!!), spammy words (FREE, BUY NOW)
- **Use:** Curiosity, vulnerability, personalization, specificity
- **Test:** Question vs statement, emoji vs no emoji, personalization vs generic

### Preview Text
- **Length:** 85-100 characters
- **Purpose:** Extend the subject line, add context
- **Avoid:** Repeating subject line exactly
- **Use:** Complete the thought, tease the value

### Sender Name
- **Default:** "Jade from Hello Little Sleepers"
- **Alternative:** "Jade + Jess (HLS)" for co-founder emails
- **Avoid:** Generic "Hello Little Sleepers" (less personal)

### Personalization Tokens
- **Use sparingly:** [First Name] in greeting (if available)
- **Avoid:** Over-personalization (creepy, not helpful)
- **Fallback:** "Hi there" or "Hi friend" if no name

### Voice Consistency
- **Warm, not corporate:** "We're here for you" not "Our team is available"
- **Conversational, not formal:** "Here's the thing" not "It is important to note"
- **Vulnerable, not perfect:** "I struggled with this too" not "We have the solution"
- **Action-oriented, not passive:** "Try this tonight" not "This could be tried"
